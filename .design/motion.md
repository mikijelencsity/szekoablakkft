# Motion Principles

## Core idea
Motion is not decoration. Motion is attention guidance.

If motion does not help understanding or decision-making, it should not exist.

## Timing system
- Fast micro-interactions: 150–250ms
- Standard transitions: 300–600ms
- Storytelling transitions: 600–1200ms

## Easing
Primary easing:
- power3.out (GSAP standard feel)
- smooth deceleration preferred over linear motion

Avoid:
- overly bouncy animations
- chaotic multi-direction motion

## Scroll behavior
Scroll should feel:
- stable
- predictable
- slightly cinematic

Use scroll-triggered motion only for:
- revealing structure
- emphasizing hierarchy
- guiding focus

## Allowed animation types
- fade + translate (default)
- scale (emphasis only)
- clip/reveal masks (premium feel)
- parallax (very subtle)
- sticky transitions (Apple-style storytelling)

## Forbidden patterns
- multiple competing animations at once
- constant motion on screen
- distracting background animations

## Section transitions
Each section transition must:
- clarify a change in topic
- not just visually separate content

## Interaction principle
Motion should answer:
"What should I look at next?"
